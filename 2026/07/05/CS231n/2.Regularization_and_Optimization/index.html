Regulation and Optimization

Loss Function : tell u how bad the model is

正则化

在Loss Function上加上 λR(W) -> 正则项
goal：prevent the model doing too much on training data
不希望model过拟合

λ -> 超参数 ， 正则化强度

L2 Regulation R(w) : W^2(每个权值平方相加) L2喜欢分散的权值

L2 R：如果权值很小，R(w) 结果就相当低

L2 正则更喜欢小的但不接近0的权值

L1 正则喜欢0的权值(稀疏向量)

可能和导数相关?

正则化可以让我们表达对模型分布的偏好，简化模型(不一定),比如dropout能让我们得到较好的效果，但是模型会更复杂

优化:得到最好的W

following the slope,going downhills

Homework1 : calc the ▽W L

Gradient descent (梯度下降)

到后面grad变小，长度也变小

停止：

  1. 固定迭代次数
  2. 最小步长

Stochastic Gradient descent

problems

  1. jittering(抖动)
  2. 局部最小 , 鞍点(梯度为0)

    在维度变高时，鞍点会变多

solution

  1. SGD + Momentum (it can converge faster)

ρ 一般取 0.9 和 0.99, 反正很大

RMSProp

  1. RMSProp

对梯度变化剧烈的方向，减小其学习率；对梯度变化平缓的方向，放大其学习率。

  1. Adam

Adam = SGD + Momentum + RMSProp

Choose the learning rate

change the learning rate:

  1. Cosine Learning Rate

  2. 迭代几次后learning rate 下降(比如变为原来1/10)

  3. Linear Warmup：先

Learning Rate Scaling Law:如果batch的数据上升N倍，learning rate 也应该上升N倍
