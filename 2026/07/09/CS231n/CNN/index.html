CNN

Perceiving and Understanding the virtual world

Convolution Layer || Pooling layer

之前的工作：
raw picture -> feature representation -> Neural Network -> label output

怎么把feature representation输入NN：

  1. 把feature的向量全部连接在一起

直接NN VS feature后NN

一般来说，用直接NN会更好

但是我们还需要人工设计design计算图->layer的连接样式

CNN结构

用途：几乎所有视觉问题，包括TEXT2IMG IMG2TEXT , 视觉分割

Fully Connected layer: templates , 拉伸

Convolution layer: 不拉伸 , filter: pixel image ,

Exp:

32 * 32 * 3 image — 5 * 5 * 3 filter -> 28 * 28 * 1 result

用 6 个 filter

3* 32 * 32 image — 6 * 3 * 5 * 5 -> 6 * 28 * 28 + bias (6 维)[每个filter公用一个bias，这个bias在一个filter中是一个标量，所以bias总共是6维的] -> 6* 28 * 28 result

bias 和 6-d的filter都是通过梯度下降得到的

Filter size:
cout * cin * Kw * Kh

在Convolution layer中间，我们需要加入Relu等其他激活函数->不然就会退化成一层convolution layer

第一层filter最终学习到的样式：

  1. 对比色 （如pink && green）
  2. 边缘检测

Q:为什么滤波器会不同
A:因为随机初始化不同，所以随机的初始时很重要的

后面的filter不能直接看出来，要看对哪些图像响应明显

Input : x * x

Filter y * y

Output (x-y+1) * (x - y + 1)

想要让OUTPUT也保持x * x

  1. paddle ，在外面一圈加入额外的数字 0

感受野

越到后面，每一个pixel表示的感受野越大

但是我们想让感受野变大变得更快

  1. stride
  2. pooling

stride , 每一次filter跳的步数

input W filter K padding P Stride S
output = (W-K+2P)/S + 1 = (W/S) + 1

Pooling

Exp:

64 * 224 * 224 -pool-> 64* 112 * 112

一般使用stride = 2 || 4

downsampling method:

  1. maxpooling
  2. meanpooling

平移不变性

先平移再卷积和先卷积再平移得到的结果一样

所以CNN是尊重空间结构的
