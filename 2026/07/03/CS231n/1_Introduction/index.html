最近邻居分类(Nearest Neighbors Classifacation)

计算距离，我们需要一个Distanc Metric(pic1, pic2)

L1 distance -> 逐像素相减取绝对值

L2 distance

最简单的实现L1

张量变换
self.Xtr = [N,D] N-数量 D-特征量

可以为[x_size, y_size, channels] 展平后的 [x_size * y_size * channels]的一维

self.Ytr = [N,]

X = [N,D]

X[i,:] = [D,]

Xtr - X[i,:] = [N,D] - [D,] 广播后为 [N,D] - N * [,D] = [N,D]

np.sum(axis = 1) = [N,D] (axis = 0,axis = 1) = [N,]

distance = [N,]

时间复杂度
TRAIN O(1)

PREDICT O(N) # 预测一个样本的复杂度

问题

我们需要在Predict降低复杂度，Train复杂度升高是OK的

1-nearest neighbors

会出现噪点 ， 比如前面的黄色

所以我们用K-Nearest Neighbors

KNN

根据这个划分空间，白色区域是相等可能的区域，可能属于不同的labels

特征有特定的意义,我们想要保存这个特征：L1更好(对特征值的选取更敏感)

特征是任意的，可以旋转：L2更好(划分边缘更平滑)

hyperparameters （超参数）

比如 KNN 中的 K 和 distance function

我们需要找到最佳参数 (tuning) 调参

  1. 选择对training data最好的参数 (x)

    K = 1 永远是最佳的

  2. 选择对test data最好的参数 (x)

    可能只对测试集有用

  3. 将training data 一部分改为validation验证data，用train训练，来后在validation上找到最好的参数，最后test，不要用test data 调参
  4. 交叉验证(每个fold都验证一遍)
