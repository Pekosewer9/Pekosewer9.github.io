3_NN_BackPropagation

Neural Networks

每个第二层的可以成为一个被共享所有类共享的模板

max(W1x,0) -> 一个简单的激活函数

如果没有激活函数，就是linear的，W = W1 * W2

max(W1,0)被称为 Relu

合适的零中心值 和 平滑度 可以获得更快的收敛速度

缺点:可能创造死的神经元

Sigmoid: 1/1+e^x SiLU x* sigmoid(x)

缺点:会把数值压缩在一个较窄的范围内，又是会让梯度消失，所以我们不会把他们用在NN的中间层

3layers - FULLY CONNECTED

我们选择正则化项而不是减少神经网络层数来防止过拟合

原因：我们要先确保神经网络理解我们的数据，确保表达能力

如L2 -> 不让W参数过大

Backpropagation

we should know how to compute dL/dW1 dL/dW2
